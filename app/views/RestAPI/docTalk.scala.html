@Messages("shortName") API

Documentation for "talk"

Returns a JSON object that describes a Talk.

  • id*: Unique ID for this talk, used everywhere, in our printed program, on the web site, etc.
  • title*: Title of this talk
  • talkType*: talk format (translated to French or English, depending on your "Accept-language" Header)
  • track*: The track label (translated to French or English, depending on your "Accept-language" Header)
  • lang*: "en" if the presentation is in English, "fr" if it's in French, etc.
  • summary*: short description of this proposal, might use markdown syntax.
  • summaryAsHtml*: HTML version, markdown transformed to HTML
  • speakers: an array of links to each speaker, with its full name
*: indicate a mandatory field, cannot be null or blank

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showTalk(Conference.currentConference.eventCode, "BBV-277").absoluteURL()
