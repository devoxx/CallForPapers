@Messages("shortName") API

Documentation for "speaker"

Returns a JSON object that describes a Speaker.

  • uuid*: Unique String identifier
  • bio *: plain text with Markdown syntax, to descibe speaker's profile
  • bioAsHtml*: bio as HTML
  • firstName*: First name
  • lastName*: Family name
  • avatarURL: URL to a photo
  • company: Company name
  • blog: Company name
  • twitter: Twitter handle
  • lang: Company name
  • acceptedTalks: an array of links to each accepted talk presented by this speaker
*: indicate a mandatory field, cannot be null or blank

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showSpeaker(Conference.currentConference.eventCode, "25b6eeb75c18e3465d5cddf2be297b8863006551").absoluteURL()
