@Messages("shortName") API

Documentation for "proposalType"

Returns the different kind of conference proposed during this conference.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showProposalTypes(Conference.currentConference.eventCode).absoluteURL()

Result:

{ "content": "All types of proposal", "proposalTypes": [ { "id": "conf", "description": "Conference, 1h, Wednesday, Thursday and Friday morning", "label": "Conference" }, { "id": "uni", "description": "Deep Dive, 3h, Monday or Tuesday", "label": "Deep Dive" }, { "id": "tia", "description": "Tools in Action, 30mn, only Monday or Tuesday", "label": "Tools-in-Action" } }
}
