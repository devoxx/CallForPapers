@Messages("shortName") API

Documentation for "list-of-speakers"

Returns a JSON Array of short description for each speakers.

  • uuid: Unique String identifier
  • fisrtName: first name
  • lastName: Family name
  • links: A Link to the Speaker object
  • avatarURL: URL to a photo

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showSpeakers(Conference.currentConference.eventCode).absoluteURL()
