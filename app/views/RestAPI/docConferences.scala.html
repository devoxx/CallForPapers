@Messages("shortName") API

Documentation for "conferences"

Returns a JSON Array of Link object, each link is a reference to a conference.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showAllConferences().absoluteURL()

Result:

{
    "content": "All conferences",
    "links": [
        {
            "href": "@ConferenceDescriptor.current().conferenceUrls.cfpURL/api/conferences/@ConferenceDescriptor.current().eventCode",
            "rel": "@ConferenceDescriptor.current().conferenceUrls.cfpURL/api/profile/conference",
            "title": "See more details about @Messages("longName")"
        },
        {
        "href": "@ConferenceDescriptor.conference2016().conferenceUrls.cfpURL/api/conferences/@ConferenceDescriptor.conference2016().eventCode",
        "rel": "@ConferenceDescriptor.conference2016().conferenceUrls.cfpURL/api/profile/conference",
        "title": "See more details about @Messages("CONF.title")"
        }
    ]
}
}
